## shortcut

+ `Alt + Enter`  导入包
+ `Ctrl+Alt+L`  格式化代码
+ `Ctrl + space` 代码提示
+ `Ctrl + alt + left` 返回上次代码
+ `ctrl + B` = `ctrl + click` 跳到定义
<!-- 查找 -->
+ `ctrl + N` 查找类
+ `ctrl + shift + N` 查找文件
+ `alt+F7` 查看类的使用 
+ `ctrl + Q` 查看说明文档
+ `ctrl + F12`  查看当前代码summary
