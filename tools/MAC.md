## shortcut

+ `option + cmd + esc` 可以结束进程
+ `cmd + N` terminal 得到焦点后执行此命令打开新的terminal
+ finder
    - `cmd + shift + G` 打开输入路径面板
    - `cmd + shift + C` 打开电脑
    - `cmd + shift + H` 打开个人目录
    - `cmd + shift + D` 打开桌面
    - `cmd + shift + A` 打开程序

## terminal

+ `open /`  `open ~`
+ `cd/ls/echo/clear`
+ `pwd` 查看当前路径
+ `ls -R`
+ `man <cmd>` help
