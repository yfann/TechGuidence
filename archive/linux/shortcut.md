## 光标移动
+ `ctrl + a/e` 行首/行尾

+ `ctrl + b/f` 左右箭头

+ `Alt + b/f` 词首词尾

## 编辑

+ `ctrl + d` 删除光标位置的字符

+ `ctrl + w` 删除光标前的单词

+ `Alt + l/u` 大小写（光标到词尾）

## 补全

+ `tab + tab`


## history

+ ~/.bash_history
    + `echo $HISTFILE`

+ `ctrl + r` 搜索历史

+ `!string`
    + `!echo` 找到最近执行过的echo开头的命令并执行

+ `!number` 执行对应命令

+ `history`
    + `history -c` 清除命令
