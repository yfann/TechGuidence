# Docker Compose
+ 通过使用 Docker Compose，您可以将多个 Docker 容器定义为一个单独的应用程序，并通过一个配置文件来定义它们之间的关系、网络设置、卷挂载等配置。然后，您可以使用一个命令来启动整个应用程序，而不必手动运行多个 Docker 命令来启动各个容器。


## cmd
+ `docker-compose up -d`
    + 目录下寻找`docker-compose.yml`

+ `docker-compose down`
