## script

+ `set [var] to [value]/[expression] <as number>`
+ `beep [num]`
+ `display dialog [var]`
+ `delay [seconds]`

## 脚本编辑器

+ `cmd + k`编译
+ `cmd + r`运行
+ `ctrl + click` tips
+ `cmd + shift + o` 打开脚本词典
