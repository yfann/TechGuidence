
## tips

+ linux单机文件系统
    + ext4
    + xfs

+ Hadoop HDFS
    + MDS(MetaData Service)
        + 单点（一致性）
    + DSS(Data Storage Service)
        + 多个

+ MooseFS
    + 多个单机文件逻辑上聚合
    + 弱一致性

+ GlusterFS
    + 无元架构,没有独立的MDS
    + 使用一致性哈希算法定位元数据和数据
+ CephFS
    + RADOS
        + 存储元数据和数据
    + 多个MDS
