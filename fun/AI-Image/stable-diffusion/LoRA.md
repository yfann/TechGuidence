# LoRA(Low-Rank Adaptation of Large Language Models)

+ 8G显存 训练
+ 200M 存储

+ 和checkpoint一起使用，在某个方面微调

## 使用
+ 下载lora模型
    + models/lora

+ prompt
    + `<lora:{lora文件名}:0.5>`

+ additional network(插件)
