## view
+ 视图切换
    + 点击导航器的`x,y,z`
    + 打开视图选择器
        + `

+ 打开sidebar
    + `n`
        + 查看坐标

+ 微调面板(move)
    + 移动后左下出现


+ 点线面模式
    + 左上角(object Mode)->edit Mode->点/线/面 mode

+ 透视模式
    + x-ray(右上)
    + 可以选中被遮挡的点线面
    + `alt + z`

+ 打开统计数据
    + overlays(右上)->statistics


+ 局部视图
    + 选中物体+ `/`

+ 正交视图（右侧坐标系下的网格图标）
    + 默认透视图（近大远小）
    + 正交视图一样大

+ 裁剪视图
    + view(左上)->view regions->clipping region

+ 窗口合并拆分
    + 鼠标落在3d视图窗口的边缘变成十字光标

        + 窗口左上改属性
