## env(python虚拟环境)

+ `python3 -m venv /path/to/new/virtual/environment`
    + 创建虚拟环境

+ ` venv\Scripts\activate`
    + 激活虚拟环境
    + powershell: `./Scripts/Activate.ps1`

+ `deactivate` 退出

## python-dotenv
+ 存储环境变量
+ `.env`
+ settings.py

## requirements

+ `pip freeze > requirements.txt`
+ `pip install -r requirements.txt`
