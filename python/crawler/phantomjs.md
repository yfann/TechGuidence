
## what

+ 虚拟浏览器,webkit引擎

## install
+ [phantomjs](http://phantomjs.org/download.html),配置bin到环境变量
+ `npm install phantomjs -g`
+ `phantomjs --version`

## cmd

+ `phantomjs` ctrl+c 退出
+ `phantomjs <xxx.js>`
+ webpage
```js
var webPage = require('webpage');
var page = webPage.create();
```
