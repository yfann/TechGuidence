# python command line

## python cmd
+ `python -m webbrowser https://pym.dev/p`
+ `python -m http.server`
    + 启动web服务器，以当前目录作为root
+ `python -m json.tool <json file>`
    + 显示json格式

## tips
+ `if __name__ == "__main__"`
    + py script直接运行，而非module导入

+ `python -m <module name>`
    + 以模块方式运行
+ `python xx.py`
    + 直接运行脚本
    + `__name__ == "__main__"`
+ `py -m uuid`
