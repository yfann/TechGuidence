
## install 3rd libs

Packages: [pypi](https://pypi.python.org/pypi)

管理工具:

+ [pip](http://pypi.python.org/pypi/pip#downloads)
    `pip install [libname]`

+ [easy installer](http://pypi.python.org/pypi/setuptools)
    `easy_install [libname]`

+ [wheel](http://pythonwheels.com/)
    `pip install *.whl`
