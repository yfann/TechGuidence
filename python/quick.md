# quick python

+ `jupyter-lab`
    + `Esc` command mode
        + `Enter` edit mode
    + `dd` delete cell
    + `a` new cell
    + `shift + up/down` select multipe cells
        + `shift + M` merge
    + `ctrl + shift + -` split 光标所在处

+ jupyter-lab command
    + `history`
    + `%lsmagic`
        + `%env xxx=xxx`
            + `%who` global vars
        + `%run ./two-histograms.ipynb`
        + ` %load ./hello_world.py` inert code
        + `%store <var>`
            + `%store -r <var>` notebook直接传递变量



+ `help()`
    + `help(<module>.<function>)`

+ `dir(<module>)`

+ `?<var>` module,method
