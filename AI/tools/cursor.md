## cmd
+ `ctrl/cmd + y` use selected
+ `ctrl + l` chat
    + `@Folders`
    + `@Git`
    + `@Codebase`

+ `ctrl[cmd] + k` generate
    + `@File`
    + `@Code`
    + `@Web`
    + `@Docs`
    + `@Definitions`
    + `@Chat`

<!-- Composer -->
+ `ctrl[cmd] + k`
    + 选中或不选中代码
    + 局部修改
+ `ctrl[cmd] + shift + k`
    + 跨多个文件的修改
+ `ctrl[cmd] + j`
    + 终端
    + shell命令

+ `ctrl[cmd] + i`
+ `ctrl[cmd] + shift + i`
