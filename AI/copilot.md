# copilot

+ `ctrl + i`/`cmd + i`
    + 光标处弹出输入框
+ `ctrl + enter`/`cmd + enter`
    + 接收建议并换行

## cmd
+ `@workspace #file:'file name.xxx' #file:'second file.xxx'`
    + provide multiple files

+ `@terminal #terminalLastCommand`
