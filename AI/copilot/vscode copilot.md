# copilot

## shortcut
+ `ctrl + alt + i`
    + 打开chat
+ `ctrl + i`/`cmd + i`
    + 光标处弹出输入框
+ `ctrl + enter`/`cmd + enter`
    + 接收建议并换行
+ `alt + ]`/`alt + []`
    + 切换solution
