# screen
+ `ssh ....`
    + `screen`
    + `while true; do date;sleep 1;done`
    + `ctrl + a + d` 退出screen
    + `screen -ls`
    + `screen -x` back to screen

+ `screen -x` back to screen
    + `screen -x [prefix name]` for multiple screen

+ `screen -S "[tag]"`
    + `screen -x [tag]`

+ `screen -L` log

+ `exit` terminate screen
