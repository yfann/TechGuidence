
## SElinux
+ 最大限度地减小系统中服务进程可访问的资源
+ 对每个进程的权限的细粒度控制
    + 传统Linux只有root用户和普通用户
+ DAC(自主访问控制)
    + 没有SElinux的系统，rwx是访问资源的权限
+ MAC(强制访问控制)
