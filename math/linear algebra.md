## Linear Algebra

+ 向量是连接数字和图形的桥梁
+ 任何向量都是由基向量组成的
+ 向量空间span=av1+bv2
+ v3=av1+bv2 线性相关
+ 向量空间的基是线性无关的


## Linear transformation

+ 线性变换可以想象成，原点不同，网格线平行且等分的变换。整个原坐标系的变换，原基向量随着目标向量一起变换，基向量的变换结果是2X2的矩阵
+ 变换可以看做向量的运动


## determinat of transformation(线性变换的行列式)

+ base vector(i,j)围成的单元正方形
+ 行列式就是单元正方形改变的面积倍数，负数表示i在j的左手边(原始j指北，i指东)
+ 向量空间的图形等比缩放
