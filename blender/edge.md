# blender edge

## shortcut
+ 删除
    + `x`
        + 删除顶点

+ 融并边
    + `ctrl + x`

+ 细分
    + 选中边 -> RMB -> subdivide

+ 滑移
    + 选中边 -> RMB -> edge slide
    + `g g`

+ 挤出
    + 边挤出为面
    + `e z`z轴方向挤出
    + 选中左侧挤出工具->左上点击 xyz(轴向挤出)

+ 倒角（bevel）
    + 边，顶点都能倒角

+ 环切(loop cut)
    + 左侧工具栏loop cut
    + LMB按下确认
    + LMB按住不放滑移
