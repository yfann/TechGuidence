## 基本操作

+ 平移
    + `shift + 中键`

+ 移动
    + `g` 自由移动
        + `g + x` 在x轴移动

+ 旋转
    + `r`
        + `r + x`
+ 复位
    + `alt + g`
        + 坐标归零
    + `alt + r`
        + 选择归零
    + `alt + s`
        + 缩放归零
