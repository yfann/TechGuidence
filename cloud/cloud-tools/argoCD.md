

## install
<!-- linux -->
+ `curl -sSLO  https://github.com/argoproj/argo-cd/releases/latest/download/argocd-linux-amd64`
+ `sudo chmod +x argocd`
+ `sudo mv argocd /usr/local/bin/argocd`

## cmd
+ `argocd login <ARGOCD_SERVER>`
+ `argocd cluster add CONTEXT [flags]`
