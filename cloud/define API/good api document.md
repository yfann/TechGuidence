## compare

+ `gitbook:` 基于 nodejs,支持导出成 pdf/epub 等格式，使用 markdown 书写
+ `mkdocs:` 基于Python，使用 markdown 书写
+ `sphinx:` 基于 Python，可以用 rst/markdown 书写，很多知名 py 项目的选择。可以基于 docstring 生成文档，支持生成 pdf/epub 等，功能相比 gitbook/mkdocs 更加强大
