## Cmd

+ `$x('//a')` 运算xpath

+ `$('a')` `$('a',context)` `$('a').href` 调用document.querySelector(),返回第一个匹配的元素

+ `$$('a')` 运算css selector,调用document.querySelectorAll(),返回所有匹配的元素

+ `$_` 取上次表达式的结果

+ `$0` 取选过的dom元素,$0~$4
