# bash


## bash info
<!-- 选择shell -->
+ `echo $SHELL`
    + 查看默认shell
+ `cat /etc/shells`
    + 查看所有shell
+ `ps`
    + 倒数第二行，当前shell
+ `which sh`
    + `chsh -s /usr/bin/sh`
    + 切换shell为sh
+ `bash`
    + 进入bash环境
    + `exit`
        + 退出
