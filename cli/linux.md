## sort

+ `sort <file>`
+ `sort -r <file>`
    + reverse order
+ `sort -n <file>`
    + sort numerically
+ `sort -nr <file>`
    + sort numerically and reverse order
+ `sort -k <column number> <file>`
    + sort a particular column
+ `sort -u <file>`
    + eliminate the duplicate entries
+ `sort -c <file>`
    + 返回第一条不在顺序的记录
+ `sort -M <file>`
    + 按月份排
+ `sort <file1> -n <file2>`
+ `sort <file> > <output>`
    + `sort -o <output> <file>`
    + 输出结果到文件
